🎯 Επισκόπηση

Το Conference Timer είναι ένα επαγγελματικό σύστημα χρονομέτρησης που σχεδιάστηκε ειδικά για παρουσιάσεις, συνέδρια και εκδηλώσεις. Προσφέρει:

⏰ Ακριβής Χρονομέτρηση

Χρονομέτρηση σε πραγματικό χρόνο με υπερπρογραμματισμό (overtime) για την παρακολούθηση παρατάσεων.

📱 Πολλαπλές Συσκευές

Έλεγχος από tablet, smartphone ή laptop με real-time συγχρονισμό μεταξύ όλων των συσκευών.

📺 Professional Display

Καθαρή, ευανάγνωστη εμφάνιση με υποστήριξη fullscreen και προσαρμοστικό μέγεθος γραμματοσειράς.

🎨 Προσαρμοσιμότητα

Λογότυπα, χρώματα, τίτλοι και μηνύματα πλήρως προσαρμόσιμα για τις ανάγκες σας.

📡 Live Event Timeline

Διαχείριση χρονοδιαγράμματος εκδηλώσεων με event markers και real-time tracking.

🔊 Ηχητικές Ειδοποιήσεις

Προσαρμόσιμοι ήχοι για προειδοποιήσεις και λήξη χρόνου.

💿 Εγκατάσταση

🖥️ Desktop Application (Συνιστάται)

  1. Κατεβάστε το αρχείο timer.exe από τον φάκελο του έργου
  2. Εκτελέστε το αρχείο ως διαχειριστής (right-click → "Run as administrator")
  3. Περιμένετε 4-5 δευτερόλεπτα για την εκκίνηση του embedded server
  4. Το Admin Panel θα ανοίξει αυτόματα στον browser σας

🌐 Web Browser (Εναλλακτικά)

  1. Εκτελέστε το server.js με Node.js
  2. Ανοίξτε browser και πηγαίνετε στο http://localhost:3000
  3. Θα ανακατευθυνθείτε αυτόματα στο Admin Panel

💡 Συμβουλή

Η desktop εφαρμογή περιλαμβάνει όλες τις δυνατότητες συμπεριλαμβανομένου του HDMI timer display για δεύτερη οθόνη!

🚀 Πρώτα Βήματα

⚙️ Βασική Ρύθμιση

  1. Ορίστε τον χρόνο: Στο μπλε πλαίσιο, εισάγετε ΩΡ:ΛΕΠ:ΔΕΥ και Προειδοποίηση
  2. Πατήστε "Εφαρμογή" για να ενημερωθεί ο χρονομετρητής
  3. Προσαρμόστε τον τίτλο: Εισάγετε τον τίτλο της παρουσίασής σας
  4. Ελέγξτε την προεπισκόπηση: Βεβαιωθείτε ότι όλα φαίνονται σωστά

▶️ Εκκίνηση Παρουσίασης

  1. Ανοίξτε το Timer Display: Κλικ στο κουμπί "🖥️ Timer HDMI" για δεύτερη οθόνη
  2. Πατήστε "Έναρξη" για να ξεκινήσει η αντίστροφη μέτρηση
  3. Παρακολουθήστε: Το χρώμα αλλάζει σε πορτοκαλί (προειδοποίηση) και κόκκινο (λήξη)
  4. Χρησιμοποιήστε τα κουμπιά +/- για προσαρμογές κατά τη διάρκεια

⚠️ Σημαντικό

Για βέλτιστη εμπειρία, χρησιμοποιήστε δεύτερη οθόνη (HDMI/DisplayPort) για το timer display και κρατήστε το admin panel στην κύρια οθόνη σας.

⚙️ Admin Panel

🎛️ Κύρια Χειριστήρια

⏰ Ρυθμίσεις Χρονομέτρου

Το μπλε πλαίσιο στο πάνω μέρος περιλαμβάνει:

  • ΩΡ/ΛΕΠ/ΔΕΥ: Ορισμός χρόνου παρουσίασης
  • Προειδοποίηση: Χρόνος σε δευτερόλεπτα για την πορτοκαλί προειδοποίηση
  • Κουμπί Εφαρμογή: Ενημέρωση των ρυθμίσεων

🎮 Χειριστήρια Timer

Κάτω από την προεπισκόπηση:

-10m -5m -1m Έναρξη Παύση Επαναφορά +1m +5m +10m

💬 Διαχείριση Μηνυμάτων

  1. Γράψτε μήνυμα στο κουτί κειμένου
  2. Επιλέξτε:
    • Ενημέρωση Μηνύματος: Στέλνει το μήνυμα στην οθόνη
    • Απόκρυψη/Αποστολή: Εναλλαγή ορατότητας
  3. Αποθήκευση: Κλικ "Αποθήκευση Μηνύματος" για μελλοντική χρήση
  4. Flash Μήνυμα: Κάνει το μήνυμα να αναβοσβήνει για προσοχή

📡 Live Event Timeline

Η δεξιά πλευρά του admin panel εμφανίζει:

  • Τρέχον Γεγονός: Ποιο event εκτελείται τώρα
  • Επόμενο Γεγονός: Το αμέσως επόμενο event
  • Μεθεπόμενο: Το event που ακολουθεί
  • Συνδεδεμένες Συσκευές: Ποιες συσκευές είναι online

📺 Timer Display

🖼️ Στοιχεία Οθόνης

📝 Τίτλος

Εμφανίζεται στο πάνω μέρος. Προσαρμόζει αυτόματα το μέγεθός του βάσει του μήκους.

⏱️ Χρονομετρητής

Κεντρικά στην οθόνη. Αλλάζει χρώμα: Λευκό → Πορτοκαλί → Κόκκινο → Κόκκινο αναβοσβήνων (overtime).

📊 Progress Bar

Κάτω από τον χρονομετρητή. Εμφανίζει γραφικά τον υπολειπόμενο χρόνο.

💬 Μηνύματα

Εμφανίζονται κάτω. Προσαρμόζουν αυτόματα το μέγεθος και μπορούν να αναβοσβήνουν.

🎨 Προσαρμογές Εμφάνισης

Από το Admin Panel μπορείτε να προσαρμόσετε:

  • Λογότυπα: Έως 6 θέσεις (γωνίες + κέντρα πλευρών)
  • Χρώματα φόντου: Επιλογή από παλέτα ή HEX κώδικα
  • Μέγεθος τίτλου: Κουμπιά +/- για προσαρμογή
  • Μέγεθος λογότυπων: Προσαρμοστικό μέγεθος

✨ Προηγμένα Χαρακτηριστικά

📱 Πολλαπλές Συσκευές

Συνδέστε tablet, smartphone ή laptop στο ίδιο WiFi δίκτυο:

  1. Σαρώστε το QR Code που εμφανίζεται στο admin panel
  2. Ή χρησιμοποιήστε τις IP διευθύνσεις που εμφανίζονται
  3. Όλες οι συσκευές συγχρονίζονται αυτόματα σε πραγματικό χρόνο

🎵 Ηχητικές Ειδοποιήσεις

Το σύστημα παίζει διαφορετικούς ήχους:

  • Προειδοποίηση: Ήχος όταν φτάνει στον χρόνο προειδοποίησης
  • Τέλος χρόνου: Διαφορετικός ήχος στο 00:00
  • Ρύθμιση έντασης: Από τις ρυθμίσεις (0-100%)
  • Απενεργοποίηση: Checkbox για πλήρη σίγαση

⚡ Flash Alert

Κάνει την οθόνη να αναβοσβήνει για έκτακτες ανακοινώσεις:

  • Manual Flash: 3 φορές και σταματάει
  • Συνδυασμός με μήνυμα: Αναβοσβήνει μαζί με το κείμενο
  • Άμεση διακοπή: Κουμπί "Διακοπή Flash"

📅 Event Timeline

Διαχειριστείτε το πρόγραμμα της εκδήλωσής σας:

  1. Ορίστε χρονοσειρά: Στις ρυθμίσεις, βάλτε ώρα έναρξης και λήξης
  2. Προσθέστε events: Κλικ στη timeline bar για νέο marker
  3. Επεξεργασία: Double-click σε marker για αλλαγή τίτλου
  4. Διαγραφή: Right-click σε marker για διαγραφή
  5. Live tracking: Η κόκκινη γραμμή δείχνει "τώρα"

💾 Αποθήκευση & Επαναφορά

Το σύστημα αποθηκεύει αυτόματα όλες τις ρυθμίσεις:

  • Auto-save: Κάθε αλλαγή αποθηκεύεται αμέσως
  • Export/Import: Αποθήκευση σε αρχείο JSON
  • Clear All Data: Επαναφορά στις προεπιλογές
  • Προφίλ εκδηλώσεων: Αποθήκευση διαφορετικών configurations

🔧 Αντιμετώπιση Προβλημάτων

🚫 Συνήθη Προβλήματα

❌ Δεν ανοίγει το Admin Panel

  1. Περιμένετε 5 δευτερόλεπτα μετά την εκκίνηση του timer.exe
  2. Ελέγξτε το firewall: Επιτρέψτε την εφαρμογή
  3. Δοκιμάστε χειροκίνητα: Ανοίξτε browser στο http://localhost:3000
  4. Restart: Κλείστε και ξανανοίξτε την εφαρμογή

📱 Δεν συνδέονται άλλες συσκευές

  1. Ίδιο WiFi: Βεβαιωθείτε ότι όλες οι συσκευές είναι στο ίδιο δίκτυο
  2. Ελέγξτε IP: Χρησιμοποιήστε τις διευθύνσεις από το "📱 Συνδεδεμένες Συσκευές"
  3. Firewall: Προσωρινά απενεργοποιήστε το firewall για δοκιμή
  4. Port 3000: Βεβαιωθείτε ότι δεν χρησιμοποιείται από άλλη εφαρμογή

🖥️ Δεν λειτουργεί το HDMI Timer

  1. Συνδέστε δεύτερη οθόνη πριν πατήσετε το κουμπί
  2. Windows Display Settings: Ρυθμίστε σε "Extend" όχι "Duplicate"
  3. Δοκιμάστε διαφορετικό cable (HDMI/DisplayPort/USB-C)
  4. Restart: Κλείστε εφαρμογή, συνδέστε οθόνη, ξανανοίξτε

🔇 Δεν παίζουν οι ήχοι

  1. Ελέγξτε ρυθμίσεις: Sound Enabled ✓ και Volume > 0
  2. Browser permissions: Επιτρέψτε ήχο στον browser
  3. System volume: Ελέγξτε την ένταση του συστήματος
  4. Refresh: Ανανεώστε τη σελίδα του timer display

⚡ Performance Tips

💡 Για βέλτιστη απόδοση

  • Κλείστε άλλες εφαρμογές που τρέχουν στο background
  • Χρησιμοποιήστε Chrome ή Edge για καλύτερη συμβατότητα
  • Ελέγξτε τη σύνδεση WiFi σε επιστημονικά/εταιρικά δίκτυα
  • Κρατήστε το laptop στο charger κατά τη διάρκεια χρήσης

🆘 Επείγουσες Καταστάσεις

🚨 Κρέμασμα κατά τη διάρκεια παρουσίασης

⚠️ Emergency Backup Plan

  1. Άμεση λύση: Χρησιμοποιήστε το ρολόι του smartphone σας
  2. Quick restart: Κλείστε εφαρμογή, ξανανοίξτε, πατήστε Reset
  3. Fallback timer: Χρησιμοποιήστε το built-in Windows timer
  4. Συνεχίστε την παρουσίαση: Μην διακόψετε τη ροή για τεχνικά προβλήματα

🎓 Χρήση σε Επιστημονικό Περιβάλλον

📊 Conference & Symposium Setup

🎤 Presentation Sessions

  • 15-20 λεπτά παρουσίαση
  • 5 λεπτά Q&A
  • Προειδοποίηση στα 2 λεπτά
  • Μήνυμα "Q&A Time" στο τέλος

☕ Coffee Breaks

  • 15 λεπτά διάλειμμα
  • Countdown μέχρι την επόμενη συνεδρία
  • Μήνυμα "Next Session: [Title]"
  • 5 λεπτά προειδοποίηση

🏆 Award Ceremonies

  • 3-5 λεπτά ανά βραβείο
  • Μήνυμα με όνομα βραβείου
  • Προσαρμογή +/- για ομιλίες
  • Flash alert για σπουδαιότητα

🍽️ Lunch Sessions

  • 45-60 λεπτά διάλειμμα
  • Timeline με επόμενα events
  • 10 λεπτά προειδοποίηση
  • Μήνυμα "Please return to auditorium"

🎯 Best Practices

  1. Προετοιμασία: Test όλα τα setups 30 λεπτά πριν
  2. Backup συσκευή: Κρατήστε tablet ως εφεδρικό controller
  3. Clear visibility: Βεβαιωθείτε ότι όλοι βλέπουν το timer
  4. Gentle warnings: Χρησιμοποιήστε το Flash με μέτρο
  5. Flexibility: Έχετε τα +/- κουμπιά έτοιμα για προσαρμογές
